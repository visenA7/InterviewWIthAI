# Security Policy

## Supported Versions

We actively support and fix security issues in the following versions:

| Version | Supported |
| ------- | --------- |
| 1.x.x   | Yes       |
| < 1.0.0 | No        |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please do **not** open a public issue. Instead, please report it responsibly by contacting the project maintainers directly or opening a draft security advisory on GitHub if applicable.

We will acknowledge your report within 48 hours and work with you to analyze, fix, and coordinate a release for the issue.
